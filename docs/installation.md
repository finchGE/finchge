# Installation
 
## Installing finchGE
The easiest way to install **finchGE** is to install it using PyPI(pip). The latest released version of **finchGE** can be installed using pip:

```bash
pip install finchge
```
To directly download wheels or source, follow [this link to our PyPI page](https://pipy.org).

Alternatively, the latest version of finchGE is available on our [github repository](https://github.com/finchge/finchge)

```
git clone git://github.com/finchge/finchge.git
```

from the finchge folder that containe `setup.py`, run:

```
pip install -e .
```
